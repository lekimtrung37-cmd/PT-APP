# **App Name**: TrungFit

## Core Features:

- User Authentication: Allow users to log in to view and manage their fitness plans and personal information.
- Trainer Authentication: Enable personal trainers to log in and manage client data, create training schedules, and monitor progress.
- Admin Authentication: Admin login for user and trainer management
- Trainer Profiles: Display profiles for Kim Trung and other trainers, highlighting their expertise and specialties.
- Service Listings: Showcase available personal training services and packages.
- Client Reviews: Allow users to give service review and display previous successful transformations.

## Style Guidelines:

- Primary color: Gold (#FFD700), inspired by the user's request for a 'yellow' tone.
- Background color: Light gray (#D3D3D3) a lighter variation of 'black' to set the color of the page background.
- Accent color: Dark gray (#A9A9A9) a darker variation of 'white' which contrast from background and brings attention to specific sections.
- Headline font: 'Space Grotesk' (sans-serif) for titles and emphasis; Body font: 'Inter' (sans-serif) for general text; pairing these is intended to provide readability with a modern feel.
- Use simple, modern icons to represent services and features.
- Design a clean and user-friendly interface that ensures smooth user experience, also keeping the user engadged and immersed into our platform.
- Incorporate subtle transitions and animations to enhance user engagement.
